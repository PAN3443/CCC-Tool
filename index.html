Welcome to the CCC-Tool! For the log in or a registration please use the buttons in the upper right corner of the window. You can use the full functionality for the creation of colormaps without an account by using the button "Anonymous User". The benefit of an account is a cloud storage for your session files and the opportunity to rate and comment colormaps from other users. For anonymous user the tool offers the export and import of a XML session file. Enjoy our tool.
Best wishes your CCC-Tool team.

The CCC-Tool

The CCC-Tool based on the idea of a general tool for the creation and export of colormaps with the effort to minimize the needed interaction components. To make this possible, we developed a colormap specification CMS, which descripe colormaps with a minimum number of key-colors. This foundation of the tool is extended by a collection of recommended colormaps (Gallery-Section), which can be adapted to encode the users own domain-specific semantics, and the facilities for analyzing colormaps. CCC-Tool supports some commonly-used color spaces and color difference metrics, currently including RGB, HSV, CIELAB, CIE94, CIEDE2000, and DIN99. For more help, you can always visit the tutorials page via the menu bar.

Custom Range:

Semantic: A custom range is defined by two values separated by comma. The range concludes with a semicolon at the end. (Example: "0.1,0.4;0.75,0.8;" results in two custom ranges) Please check that the second value of a range is alwasy bigger than the first one. Overlapping range are not allowed!

Info: The probe range is allowed to be over the borders of the CMS range. Probe value out of the CMS range will be excluded during the calculation of the probe colormap.
