Manage Your Colormaps

Welcome to the CCC-Tool. The following is a short instruduction. For a comprehensive introduction please visit the tutorial ("Go to ..." → Tutorial). The present page is the My Maps side, where you can manage your set of colormaps and handle the available options for exporting, editing, analysing and comparing colormaps. You can reach this window everytime by clicking on the "My Maps" button at the "Go to ..." menu. For a easy and fast start you can choose the option of adding an existing colormap (click on the "Add Existing Map" button). The new displayed side show you a set of common used colormaps (made available by sciviscolor.org ). Your selection will automatically tranfer to this side. Alternative to the selction of a existing colormap you can create your own colormap, which meet your special requirements (click on the "Create New Map" button). If you need help at the creator side please hover the questionmark button at the top for a short help or visit the tutorial. Start with:

Colormap Course

Info: With the analyse of the colormap course you can see how the colormap run inside the different colorspaces. Switch the colorspace to display the different colorspace visualizations.

Use the colormap course to observe that the colormap parts do not cross or run to close together (At the RGB space the 3D visualization should be helpful). At the visualizations of the HSV, Lab and DIN99 space you can also check the increase and decrease of the lightness or the value.

: Constant Band

INFO: Constant bands define one color for a certain section. Inside of the visualization, a constant band will be represented by squares.

: Scaled Band

INFO: Scaled bands define a color gradient for a certain section. Intermediate colors will be interpolate inside the selected colorspace. Inside of the visualization, a scaled band will be represented by two circles and a solid line. The two circles are showing the start and the end color. The line represent the interpolated colors.

: Jump

INFO: If a neighbour of a band has a different start (end) color as the end (start) color of this band, there is a jump between the band and his neighbour. This jump is not part of the colormap course, but it shows the connection between those bands. Inside of the visualization, a jump will be represented by a dashed line.

Analyse of the Difference Metrics

Info: The following visualizations are showing you the ratio value of the euclidean distances and the metrics (DE94, CIEDE2000) to determine the difference of the used key colors, which define the start and the end of a band, in relation to the ratio values of the references (x-values). Use the information of one of the both metrics or the euclidean distance of the DIN99 space to create a colormap as good as possible for the perceptual uniformity (Approach the selected ratio of the difference to the ratio of the reference values).

Attention: For this analyse only scaled band will be used.
